# 🔊 AI AudioGen Presets Generator

Генератор звуковых эффектов (SFX) на базе нейросети **AudioGen (Facebook/Meta)**. Проект позволяет массово создавать аудио-уведомления для разных тематик: от киберпанка и хоррора до ASMR.

## 🚀 Особенности
* **8 готовых стилей:** Army, Cyberpunk, Fantasy, ASMR, Maniac и другие.
* **GPU Ускорение:** Полная поддержка NVIDIA CUDA внутри Docker.
* **Автоматизация:** Генерация пачек звуков одной командой.
* **Умные тайминги:** 1 сек для обратного отсчета (COUNTDOWN) и 4 сек для основных эффектов.

## 🛠 Технологический стек
* **AI Model:** Meta AudioGen Medium
* **Runtime:** Python 3.10 + PyTorch (CUDA 12.1)
* **Infrastructure:** Docker + Docker Compose

---

## 📦 Быстрый старт

### 1. Требования
* Установленный **Docker** и **Docker Compose**.
* **NVIDIA Container Toolkit** (для работы GPU в Docker).
* Видеокарта NVIDIA (минимум 8GB VRAM для комфортной работы).

### 2. Сборка и запуск
Просто запустите команду в корне проекта:
```bash
docker-compose up --build
```

### 3. Результат
После завершения работы скрипта, все сгенерированные файлы появятся в папке:
`./tmp/[название_пресета]/[событие].wav`

---

## ⚙️ Настройка пресетов
Вы можете редактировать описания звуков в файле `presets.json`. Скрипт автоматически подхватит изменения при следующем запуске.

**Структура JSON:**
* `Ключ (название папки)` -> `Событие (имя файла)` -> `Описание (промпт для нейросети)`

---

## 🐳 Docker детали
* **Базовый образ:** `nvidia/cuda:12.1.0-base-ubuntu22.04`
* **Кэширование:** Модель скачивается в папку `/app/models` внутри контейнера при сборке образа, чтобы не тратить трафик при каждом запуске.
* **Volume:** Папка проекта монтируется в `/app`, что позволяет сохранять результат генерации сразу на ваш компьютер.

---

## ⚠️ Устранение проблем
Если Docker не видит видеокарту, убедитесь, что в `docker-compose.yml` указаны `reservations` для GPU, а в системе установлены драйверы NVIDIA.

Если у вас мало видеопамяти, скрипт может упасть с ошибкой `Out of Memory`. В этом случае закройте тяжелые приложения (браузеры, игры) перед запуском.
